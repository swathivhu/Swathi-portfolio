# **App Name**: SynthFolio

## Core Features:

- Fixed Profile Sidebar: Displays personal information, a circular profile image, title, intro text, social links (GitHub, LinkedIn, Email), and a glowing 'Download Resume' button in a fixed left column.
- Dynamic Content Sections: A scrollable main area presenting structured content sections: Hero, About, Skills, Projects, Experience, Education, AI Creations, and Contact.
- Floating Section Navigation: Floating navigation icons in the top-right corner provide smooth scrolling access to each main content section.
- AI-Powered Project Overviews: Utilizes an LLM tool to generate concise summaries or key takeaways for complex projects, enhancing presentation based on user interaction or context.
- Interactive Skill and Project Cards: A grid layout for skills with glowing, hover-animated cards, and interactive project cards displaying titles, descriptions, technology stacks, GitHub links, and live demo buttons.
- Contact Form Integration: A glassmorphism contact card featuring form fields for Name, Email, and Message, alongside social links for visitor inquiries.
- Responsive Design & Futuristic Styling: Ensures the portfolio is fully responsive across desktop, tablet, and mobile devices, maintaining a dark, futuristic UI with glassmorphism elements and smooth transitions.

## Style Guidelines:

- Color scheme: Dark tech theme with a primary color of vibrant electric blue-violet (#6C63FF). This provides a modern, energetic foundation against the deep, dark blue background (#0F172A), evoking a futuristic and immersive experience. A striking bright aqua (#00F5D4) serves as an accent for highlights and interactive elements, creating dynamic contrast and a 'glowing' effect.
- Backgrounds: Main background is a very dark blue (#0F172A) for depth and a premium feel. Glassmorphism cards utilize a slightly lighter, desaturated dark slate blue (#1E293B) to stand out with translucency.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, techy, and futuristic feel, ideal for titles and short, impactful text. Body text font: 'Inter' (sans-serif) provides excellent readability and a clean, neutral aesthetic suitable for longer descriptions and detailed content.
- Icons: Use sleek, minimalist line icons for navigation, social links (GitHub, LinkedIn, Email), and general UI elements. Icons should adopt the glowing accent color on hover or when active, reinforcing the futuristic theme.
- Two-column responsive layout: A fixed left sidebar for the profile and a scrollable main content area on the right. Key sections like 'About' and 'Contact' feature glassmorphism cards. 'Skills' and 'AI Creations' sections are organized in a dynamic grid, while 'Experience' uses vertical cards and 'Education' features a glowing vertical timeline.
- Animations: Incorporate smooth scrolling navigation, captivating card hover lift effects, subtle glowing borders on interactive elements, and a dynamic typing animation for the hero title. Implement scroll fade-in animations for section reveals (e.g., About section, Education timeline), along with subtle animated background particles in the hero section, a floating gradient background for general ambiance, and an interactive custom cursor to enhance user engagement.
